# 🛒 SaverScout

**Scout smarter. Spend less.**

SaverScout is an Australian grocery price transparency platform that helps shoppers:

- Compare weekly supermarket specials
- Track price history over time
- Get notified when favourite items go on sale
- See community-reported local price differences

---

## 🚀 MVP Scope

- Coles + Woolworths (TAS-first rollout)
- Weekly baseline special ingestion
- Community price observations with trust scoring
- Effective price resolver
- Email alerts for favourite products

---

## 🏗️ Tech Stack

**Backend**
- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- Docker

**Frontend**
- React + Vite (planned)
